# DoorLock
To have a door lock that can be opened remotely using 

Items:
1. 12v Adapter
2. 12v Adapter socket
3. 7805
4. Node mcu
5. TIP122
6. 12v solenoid lock, spring loaded
7. Enclosure box
8. Pcb
9. Wooden socket for solenoid lock
10. Screws or 
 1. glue or
 2. hole in the door frame
 3. Metal hinge for lock
11. Cable guide
